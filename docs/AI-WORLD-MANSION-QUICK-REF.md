## 🔧 快速配置

### 修改角色设定
```csharp
// 文件: AgentRepository.cs
// 方法: InitializeDefaultAgents()
new PersistedAgentProfile
{
    Id = "角色ID",
    Name = "角色名称",
    Avatar = "表情符号",
    Personality = "性格描述",
    SystemPrompt = "详细提示词...",
    Description = "简短描述",
    Enabled = true
}
```

### 修改路由规则
```csharp
// 文件: AgentGroupRepository.cs
// 方法: InitializeDefaultGroup()
TriageSystemPrompt = @"
路由规则：
1. 关键词1、关键词2 → 角色ID
2. 关键词3、关键词4 → 角色ID
..."
```

### 调整工具提示
```csharp
// 文件: WorkflowManager.cs
// 方法: CreateWorkflow()
instructions: profile.SystemPrompt +
    "\n\n额外的工具使用提示..."
```

## 🐛 常见问题速查

### Q1: 路由总是到同一个角色？
**A**: 检查Triage提示词，可能需要更明确的关键词区分

### Q2: 角色回复不像人设？
**A**: 检查SystemPrompt是否足够详细，可以添加更多示例

### Q3: 工具调用失败？
**A**: 检查appsettings.json中的API密钥配置

### Q4: 数据库初始化失败？
**A**: 检查文件权限，确保应用可以写入db文件

### Q5: 编译错误？
**A**: 运行 `dotnet clean` 后再 `dotnet build`
